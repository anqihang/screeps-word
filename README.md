# screeps

## 模块功能

### main:主文件

> - 角色（role)

    - role.harvester  采集者
    - role.builder    建造者
    - role.upgeader   升级者
    - role.repairer   维修者(消耗1修复40)
    - role.carrier    运输者

> - 基础工作(base work)

    - harvest   采集
    - withdraw  拿取

> - 配置文件(config.json)

    - creeps    数量，部件

> - 建筑(structure)

    - structure_tower

## 颜色标志

> - 线路

     - 红色：攻击状态<br/>
     - 绿色：建造状态<br/>
     - 蓝色：运输状态<br/>
     - 橘色：修复状态<br/>
     - 黄色：采集状态<br/>
